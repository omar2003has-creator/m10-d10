# Fork-and-submit workflow — M10 Drill

GitHub Classroom is sunset. From Module 8 onward, AISPIRE delivers
assignments via fork-and-submit.

## Steps

1. From the template repo page (`LevelUp-Applied-AI/m10-d10`) click
   **Use this template → Create a new repository** (or **Fork**).
   Choose your own account and make the new repo public.
2. Clone your fork locally.
3. Create a branch: `drill-10-typed-retrieval-endpoint`.
4. Implement the TODOs in `main.py`.
5. Push the branch and open a PR **within your own fork** (compare
   `drill-10-typed-retrieval-endpoint` against your fork's `main`).
6. Wait for the autograder to run; iterate until it passes.
7. Paste your PR URL into TalentLMS → Module 10 → Core Skills Drill to
   submit.
